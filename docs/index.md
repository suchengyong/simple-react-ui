## 轻松上手

您可以使用 npm / yarn 来安装。

```
$ npm install simple-react
或者
$ yarn add simple-react
```

## 简单使用

```
import  {  Button  }  from  'simple-react'
import  'simple-react/lib/simple-react.css'

<Button  type="primary">Primary</Button>
```
